About Me

Hi there, my name is Andrew and I’m a 25-year-old Front-End Web Dev from Ireland. From a young age I have loved to build things from gaming PCs to PVC airsoft guns. Over the last two years I have been building websites and progressive web apps using HTML CSS and JavaScript.I have a hunger to learn and develop as a person; I’m using this desire to further my knowledge in web and software development. Projects

E-commerce website

Github Project

I built this ecommerce website with the bootstrap 4 library. I decided on this as I knew taking on a large project like an ecommerce site would include a large number of media queries to get the responsiveness required. Bootstrap with its flexible grid system would help me achieve this. Included in the library is the carousel component, which is used to showcase items on sale. For the website to be functional I needed a sign in section with a form that validates the users email and checks that all input fields are completed. Currently I’m working on a cart section that accumulates all items and prices into a users cart using JavaScript.
